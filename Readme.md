# **arXivDiff**: A bash script to download and compare two papers on the arXiv

### Usage
`./arXivDiff xxxx.xxxxx`

to compare v1 and v2, or

`./arXivDiff xxxx.xxxxx v3 v5`

for some arbitrary versions.

### Dependencies
wget, latexdiff, pdflatex

### Misc
This script was inspired by Timon Emken's [comparxiv](https://github.com/temken/comparxiv).
